# Expense Analysis Project

## Objective
Analyze transaction data to identify spending patterns and key cost drivers using Excel.

## Dataset
The dataset consists of categorized expense transactions.

## Approach
- Cleaned and structured raw transaction data
- Aggregated expenses by category using conditional formulas
- Displayed percentages in a table and pie chart

## Key Insights
- Housing accounts for over half of total monthly spending.
- Food & Utilities make up about one-third of total expenses.
- Entertainment and Other expenses remain under 5%.
## Tools Used
- Microsoft Excel (SUMIF, percentage calculations, structured worksheets, charts)

## Files
- `expense_analysis_january.xlsx` – Excel workbook containing raw data, calculations, and  summary
- Images – Screenshots of key worksheets used for visualization and presentation
